# Changelog

All notable changes to Constellation will be documented in this file.

---

## [0.2.1] - 2024-05-20

### Added
- Comprehensive documentation site (MkDocs).
- Support for `SnapshotEvent` for enhanced viewer integration.
- New tutorial for building domain plugins.

### Improved
- Refined `LLMAgent` error handling for malformed JSON responses.
- Updated Supply Chain example with more realistic price elasticity.

### Fixed
- Fixed a bug in Dijkstra pathfinding where edge distances weren't always respected.

---

## [0.2.0] - 2024-04-15

### Added
- Core engine with tick/phase architecture.
- Graph-based topology with built-in pathfinding.
- SQLite persistence for simulation events.
- Initial React-based real-time viewer.
- Global Supply Chain example.
- Mini Stock Market example.
